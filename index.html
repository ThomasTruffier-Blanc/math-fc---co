Fiches rapides

Les notions du cours, condensees

Etude d'une fonction reelle

Une etude complete doit etre automatique. Tu cherches d'abord ou la fonction existe, puis les limites, les points singuliers, la derivee et le signe de la derivee.

T(x) = f'(x0)(x - x0) + f(x0)
  1. Continuite en a: lim f(x) = f(a).
  2. TVI: si f est continue sur [a,b], toute valeur entre f(a) et f(b) est atteinte.
  3. Bijection: si f est continue et strictement monotone, la solution est unique.
  4. Variation: f' positive donne f croissante, f' negative donne f decroissante.

Fonctions usuelles

Le plus important est de reconnaitre vite les derivees et les limites comparees. Exp gagne contre les puissances, ln grandit plus lentement que x.

(e^u)' = u'e^u ; (ln u)' = u'/u ; (u^alpha)' = alpha u'u^{alpha-1}
  1. e^x ~ 1 + x quand x tend vers 0.
  2. ln(1 + x) ~ x quand x tend vers 0.
  3. log_a(x) = ln(x)/ln(a) et (a^x)' = ln(a)a^x.
  4. arcsin'(x) = 1/sqrt(1-x^2), arccos'(x) = -1/sqrt(1-x^2), arctan'(x) = 1/(1+x^2).

Nombres complexes

Un complexe z = a + ib se lit dans le plan: a est la partie reelle, b la partie imaginaire. La forme polaire donne le module et l'argument.

z = a + ib = rho e^{i theta}, avec rho = |z| = sqrt(a^2 + b^2)
  1. Conjugue: z barre = a - ib.
  2. Inverse: multiplier par le conjugue du denominateur.
  3. Euler: e^{i theta} = cos theta + i sin theta.
  4. Moivre: (e^{i theta})^n = e^{in theta}.

Representation complexe des signaux

Un signal sinusoidal peut etre represente par une amplitude complexe. Cette representation simplifie les derivations, integrations et sommes de signaux de meme pulsation.

x(t) = E cos(omega t + phi) ; X(t) = E e^{j(omega t + phi)}
  1. Amplitude complexe: X = E e^{j phi}.
  2. Deriver revient a multiplier par j omega.
  3. Integrer revient a multiplier par 1/(j omega).
  4. Fresnel represente X comme un vecteur dans le plan complexe.

Entrainement guide

Exercices reflexes

Continuite

Point de raccord

Comment montrer qu'une fonction est continue en a ?

Calcule la limite a gauche et a droite en a, puis compare avec f(a). Si les trois valeurs sont egales, f est continue en a.

Tangente

Equation locale

Donner l'equation de la tangente a f en x0.

T(x) = f'(x0)(x - x0) + f(x0). Il faut donc calculer f(x0) et f'(x0).

Derivee

Composee

Deriver f(x) = e^{3x^2 - 2x}.

Avec u = 3x^2 - 2x, u' = 6x - 2. Donc f'(x) = (6x - 2)e^{3x^2 - 2x}.

Asymptote

Verticale

Quand la droite x = a est-elle asymptote verticale ?

Si f(x) tend vers +infini ou -infini quand x tend vers a, au moins d'un cote.

Ln

Limite usuelle

Calculer lim ln(1+x)/x quand x tend vers 0.

La limite vaut 1, car ln(1+x) est equivalent a x au voisinage de 0.

Complexes

Module

Calculer le module de z = 3 - 4i.

|z| = sqrt(3^2 + (-4)^2) = sqrt(25) = 5.

Euler

Forme algebrique

Mettre e^{i pi/2} sous forme algebrique.

e^{i pi/2} = cos(pi/2) + i sin(pi/2) = i.

Signal

Amplitude complexe

Pour x(t)=3 cos(1000t - 0,2), quelle est l'amplitude complexe ?

X = 3e^{-j0,2}. La partie e^{j1000t} porte l'evolution temporelle.
